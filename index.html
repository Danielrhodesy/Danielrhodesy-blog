Display Positions

Relative

Relative is a position property that is basically relative to it self. So if the coder has it already setup in a position and they add top:10 it will move from where it would normally be. Other content cannot move like this with the properties top,right,bottom,left.


Absolute

An element with the position absolute will be relative to the nearest positioned ancestor. It can be overwritten over anywhere in the page of contents. So it is very powerful as you can place this element anywhere you want it.


Fixed

This positioning it used to be relative to the viewport, the viewport is from what screen size you are using to look at the site. And not to any other element.

Differnce between HTML and CSS

Imagine if HTML is a house. A house has room furniture all solid thing which fills out this area and creates something to the user. All the header, div, body tags are all the content in this house. For this house to be pleasing and livable towards people it needs CSS which is basically the rules for the house , giving each part of the house colour and position rules to where each piece is.


Explain control flow and loops

Control flow is the scope of your order you have made in your code. It means what is going to execute in the order you have put it. In Javascript the flow goes from top to bottom. And a loop is a function witch if under the right arguments it will repeat the same function until it has exceeded the parameters you have set it to. An Analogy for this would be comparing control flow to a recipe. As you have to follow the recipe from the top to the bottom. When a loop comes up that could be you having to repeat one part of the recipe until it has told you the desired result.


Describe what the DOM

The DOM is an interface to the HTML document. It is essentially an API to the page allowing programs to read and manipulate the pages content, structure and styles. It converts the structure of the HTML to be able to be modeled by various programs. You can interact with the DOM by using javascript, as I could select, highlight or change sections in the DOM and make them have useful programs and function attaches to the different sections.


Difference between arrays and objects

Array and objects both hold data but in a different way. Arrays are more simple and hold data in a list. Like a shipping list (butter, eggs, milk, chocolate). Where objects hold data in a list but also give each data peice keys and values to those keys. Just like (butter : 100, eggs : 12) and so on.
